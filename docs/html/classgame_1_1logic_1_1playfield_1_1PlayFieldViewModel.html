This method accesses the internal representation of the play field, which is organized as a two-dimensional array. It returns the tile located at the given (x, y) coordinates. The coordinates must be within the bounds of the play field. void game.logic.playfield.PlayFieldViewModel.Update ( )

Updates the game state by processing events from the event queue.

This method retrieves the event queue from the service locator and checks for a specific event identified by EventId.SpawnTile. If a SpawnTileEvent is found in the queue, it invokes the delegate associated with the event to create a tile shape. The created tile shape is then placed at the specified coordinates (0, 0) in the game world. This method is typically called during the game loop to ensure that events are processed in a timely manner.

Implements gamerunner.IUpdatable.
