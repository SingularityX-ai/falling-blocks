void GameRunner.Update ( )
inline

Updates the state of the object by processing gravity and spawning tiles when a threshold is reached.

This method retrieves the current gravity from the GravityService and accumulates it in the _gravitySum field. It increments the frame count and checks if the accumulated gravity exceeds 100.0f. If it does, it resets the frame count and gravity sum, and spawns a new tile shape using the TileSpawnerService. It also enqueues a SpawnTileEvent to notify other parts of the system about the new tile creation. The method ensures that a new tile shape is created each time the threshold is crossed.

Implements gamerunner.IUpdatable.
