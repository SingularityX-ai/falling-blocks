void game.logic.tilespawner.TileSpawnerService.Shuffle ( )
inlineprivate

Shuffles the elements of the collection randomly.

This method implements the Fisher-Yates shuffle algorithm to randomly reorder the elements in the private collection of pieces. It iterates through the collection, selecting a random index for each element and swapping it with the current element. This ensures that each possible permutation of the collection is equally likely, resulting in a fair shuffle. The method modifies the original collection in place, meaning that the order of elements in _pieces will be changed after this method is called.
