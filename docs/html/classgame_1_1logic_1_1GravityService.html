void game.logic.GravityService.Update ( )
inline

Updates the current gravity strategy if it implements the IUpdatable interface.

This method checks if the current gravity strategy (_gravityStrategy) can be cast to the IUpdatable interface. If it can, the Update method of the updatable strategy is called. This allows for dynamic updates to the gravity strategy's state or behavior. This is particularly useful in scenarios where the gravity strategy may change over time and needs to be updated accordingly.

Implements gamerunner.IUpdatable.
